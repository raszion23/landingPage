# Landing Page

This is my solution to The Odin Project's Foundations Landing Page project.

## Overview

The goal of this project was to recreate a landing page from a provided design using HTML and CSS. This project helped me practice page layout, Flexbox, spacing, typography, colors, and structuring a webpage from a visual reference.

## Built With

- HTML
- CSS
- Flexbox

## What I Learned

During this project, I practiced:

- Building a full landing page from a design reference
- Using Flexbox to align and position sections
- Creating reusable layout patterns
- Managing spacing, margins, and padding
- Styling buttons, text, and page sections
- Organizing project files and image assets
